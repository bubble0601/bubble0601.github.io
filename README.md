# Github.io

## githubでウェブサイトを作ってみた
[URLはこちら](https://bubble0601.github.io)

## Setup
```
% git clone https://github.com/bubble0601/bubble0601.github.io
% cd bubble0601.github.io
% npm install
% npm run build
```
