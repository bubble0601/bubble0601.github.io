Main関数

エントリーポイント

例1

int main(void) {
    return 0;
}

例2(voidは省略可)

int main() {
    return 0;
}

例1ではmain(0)などとするとエラーになるが例2ではエラーは起きない

例3(一応動くが、省略すべきではない)

main(void) {
    return 0;
}

例:

int main(void) {
    printf("Hello, World!\n");
    return 0;
}

出力

Hello, World!
