main関数

Java

Javaではファイル名と一致する名前のクラスのmainメソッドから処理が始まる

Main.java

class Main {
    public static void main(String args[]) {
        /* 処理 */
    }
}

publicはアクセス修飾子でメソッドが公開されていることを意味している.

staticは静的であることを意味する修飾子. main関数はクラスメソッドでなければならない.

void返り値の型. voidなので返り値はなし.

String args[]コマンドライン引数

全て省略不可. void以外の返り値も不可.

例

Main.java

class Main {
    public static void main(String args[]) {
        System.out.println("Hello, World!");
    }
}

実行

javac Main.java
java Main

出力

Hello, World!
