main関数的なやつ

2017/12/6

ファイルの1行目から順に実行される

以下のように書くと、if文の中は直接実行された場合は呼び出されるが、モジュールとしてインポートしたときは呼び出されない

if __FILE__ == $0
    # 処理
end

例

def main
    puts "Hello, World!"
end

if __FILE__ == $0
    main
end

出力

Hello, World!
