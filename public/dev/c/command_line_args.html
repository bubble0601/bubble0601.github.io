コマンドライン引数

2018/1/12

コマンドラインから引数を受け取る

int main(int argc, char *argv[]) {
    /* argcに引数の数, argvに引数が格納されている */
    return 0;
}

以下も可

int main(int argc, char **argv) {}

例

foo.c
#include <stdio.h>

int main(int argc, char *argv[]) {
    for (int i = 0; i < argc; i++) {
        printf("%s\n", argv[i]);
    }
    return 0;
}
実行
gcc ./foo.c
./a.out bar
出力
./a.out
bar
