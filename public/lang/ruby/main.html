main関数的なやつ

Ruby

ファイルの1行目から順に実行される

以下のように書くと、if文の中は直接実行された場合は呼び出されるが、モジュールとしてインポートしたときは呼び出されない

if __FILE__ == $0
    # 処理
end

例

if __FILE__ == $0
    puts "Hello, World!"
end

出力

Hello, World!
